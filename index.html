Tweet Map

Tweet Map is a project I worked on when I was invited to Twitter to learn how to develop Android applications using Fabric.io, Twitter's own mobile development platform. It allows users to recognize the local or global impact of their tweets by projecting the geographical locations of their retweets onto a Google Maps Android Activity.

Course Surfer

Course Surfer is an application designed to streamline the registration period at Brown University. It provides users with a scheduling algorithm which curates non-conflicting schedules tailored to the users' work preferences. It also provides its users a means of monitoring the availability of capped courses, getting email and Twilio text notifications when a spot opens up for enrollment. Lastly, we aimed to socialize the registration period by providing a schedules' sharing feature, allowing users to share exciting schedules with their friends via the interface.

    • Backend & Server-side: Java | Java Spark Server
    • Database & Storage: Sqlite
  • March 2016 - May 2016
